# Visual Identity Guidelines
**Brand:** Centro Espírita Xangô da Pedreira
**Project:** ID Card Generator • Umbanda (Gerador de Carteiras • Umbanda)

## 1. Brand Concept & Motto
* **Motto (Portuguese):** "Fé, caridade e amor. São caminhos que te levam à luz."
* **Motto (English Translation):** "Faith, charity, and love. These are paths that lead you to the light."

## 2. Color Palette
The color palette uses earthy and celestial tones representing the spiritual and structural elements of the brand.

| Color Name | Hex Code | Purpose/Association |
| :--- | :--- | :--- |
| **Dark Brown** (Marrom Escuro) | `#3B1F0E` | Primary dark tone, text, high-contrast backgrounds. |
| **Medium Brown** (Marrom Médio) | `#7A4A1F` | Secondary earthy tone, accents, borders. |
| **Gold** (Ouro) | `#D4A017` | Highlights, primary action buttons, decorative elements. |
| **Beige / Pearl** (Bege/Pérola) | `#F6EBDD` | Main background color, providing a soft parchment/classic feel. |
| **Sacred Blue** (Azul Sagrado) | `#00A9C6` | Accent color, used in the logo to represent water/spiritual energy. |

## 3. Typography
The typography relies on classic, serif fonts to convey tradition, seriousness, and elegance.

* **Titles & Highlights:** `Cinzel Semibold` (Primarily used in Uppercase).
* **Body Text:** `Lora Regular`.
* **Support & Complementary Text:** `Playfair Display Italic`.

## 4. Logo & Iconography

### 4.1. Logos
* **Main Logo:** Features the brand icon placed on the left, with the text "CENTRO ESPÍRITA XANGÔ DA PEDREIRA" and the subtitle "GERADOR DE CARTEIRAS • UMBANDA" on the right.
* **Logomark (Icon):** Contains crossed axes (representing the Orisha Xangô), a book, a straw/basket element, water waves at the base (Sacred Blue), a crescent moon, and a star.

### 4.2. System Icons
The UI uses a solid, flat-icon style in Dark Brown (`#3B1F0E`):
* **Generate ID Card** (Gerar Carteira): ID badge with a user silhouette.
* **My ID Cards** (Minhas Carteiras): Stacked cards.
* **Mediums** (Médiuns): Group of three people/users.
* **About the Center** (Sobre o Centro): Open book.
* **Help** (Ajuda): Question mark inside a circle.
* **Settings** (Configurações): Gear/cogwheel.

## 5. UI Elements

### 5.1. Buttons
* **Primary / Dark Button:** Dark Brown (`#3B1F0E`) background, white text, left-aligned icon. Example: "GERAR CARTEIRA".
* **Secondary / Outline Button:** Transparent or Beige background, Dark Brown text, left-aligned icon. Example: "VER EXEMPLO".
* **Action / Highlight Button:** Gold (`#D4A017`) background, white text, left-aligned icon. Example: "ENTRAR".
* **Text Link:** "VER MAIS ->" with a solid underline in Medium Brown.

### 5.2. Patterns & Backgrounds
* **Watermark Pattern:** Subtle, low-opacity line art of the main icon used as a background accent.
* **Dark Background Card:** Dark Brown (`#3B1F0E`) background with ornate, golden floral/leaf corner borders.
* **Light Background Card:** Beige (`#F6EBDD`) textured background with ornate, dark brown corner borders.

### 5.3. Decorative Elements
* Vintage, stylized leaf and branch flourishes.
* Ornate, symmetrical line dividers with diamond-shaped center bullets.
